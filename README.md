# Student Study Analysis Project

## About
This project analyzes student performance based on
Study Hours, Sleep Hours, Attendance and Marks.

## Dataset
- 10 students
- 5 columns: Student, Study_hours, Sleep_hours, Attendance, Marks

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI
- Google Colab

## Files
- analysis.ipynb → Python code
- student_data.csv → Dataset
- study_analysis.png → Scatter plot
- student_marks_bar_chart.png → Bar chart
- students analysis dashboard.pbix → Power BI Dashboard

## Results
- Top student: Rohit with 95 marks
- Average marks: 68.7
- More study hours = Better marks
